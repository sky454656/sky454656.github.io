공격자 관점으로 배우고, 기록으로 남기는 보안 블로그

SKY LAB NOTE는 실습 기반 개발 기록과 취약점 분석 메모를 함께 쌓는 공간입니다. 단순 링크 모음이 아니라, 재현 과정과 판단 근거가 남는 글을 목표로 운영합니다.

Hands-on Development 자동화 스크립트, 분석 도구, 블로그 운영 파이프라인까지 직접 구현한 내용을 다룹니다.
Offensive Security CTF, 워게임, 버그바운티, 실제 취약점 분류 기준을 정리하며 공격 표면을 분석합니다.
Study With Output 논문, 컨퍼런스, 기술 문서를 읽고 끝내지 않고 실험이나 요약 결과물로 남깁니다.

이 블로그에서 다루는 6개 축

  • 개발: 자동화, 스크립트, 환경 구성, 작은 도구 제작 기록
  • CTF/Wargame: 문제 풀이 흐름과 취약점 재현 포인트 정리
  • BugBounty: 리컨, 우선순위, 리포트 관점에서의 정리
  • 블로그/기술문서: 문서화 방식, 정보 구조화, 운영 자동화 기록
  • 논문/컨퍼런스: 보안 연구 트렌드와 발표 내용 요약
  • 공모전/자격증: 준비 전략, 학습 로드맵, 회고
